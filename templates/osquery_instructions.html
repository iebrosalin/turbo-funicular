Установка OSquery

sudo apt update && sudo apt install osquery -y

Скопируйте osquery.conf в /etc/osquery/. Запустите: sudo systemctl enable --now osqueryd

Скачайте MSI: https://pkg.osquery.io/windows/osquery.msi
Установка: msiexec /i osquery.msi /qn. Конфиг в C:\ProgramData\osquery\osquery.conf. Запуск: sc start osqueryd
